This repo contains multiple folders / modules — each corresponds to a project:

01vitereact – React with Vite as the build tool.

02counter – Simple counter app showcasing state and events.

03tailwindprops – Demonstrates Tailwind CSS + React props.

04bgChanger – Background color changer example.

05passwordGenerator – Password generator app built with React.

06currencyConvertor – Currency converter application.

07reactRouter – React Router integration for multi-page apps.

08miniContext – A small example of Context API usage.

09themeswitcher – Theme switching (dark/light) in React.

10todocontextLocal – Todo app using Context API + localStorage.
